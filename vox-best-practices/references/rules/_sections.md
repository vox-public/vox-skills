# Rule Sections

## 1. Flow Agent Design

- **Impact**: CRITICAL
- **Prefix**: `flow-`
- **Description**: Guidelines for designing conversation flows in Flow Agent. Covers structure, branching logic, fallback handling, and context management.

## 2. Prompt Agent

- **Impact**: HIGH
- **Prefix**: `prompt-`
- **Description**: Best practices for writing effective prompts in Prompt Agent. Covers system prompts, context provision, and constraint definition.

## 3. Voice Configuration

- **Impact**: HIGH
- **Prefix**: `voice-`
- **Description**: Guidelines for configuring TTS (Text-to-Speech) and STT (Speech-to-Text) settings. Covers provider selection, latency optimization, and natural speech configuration.

## 4. API Integration

- **Impact**: MEDIUM
- **Prefix**: `api-`
- **Description**: Best practices for integrating external APIs with vox.ai agents. Covers webhook configuration, timeout settings, and retry logic.

## 5. Error Handling

- **Impact**: MEDIUM
- **Prefix**: `error-`
- **Description**: Guidelines for handling errors gracefully in vox.ai agents. Covers error recovery, logging, and user-friendly error messages.

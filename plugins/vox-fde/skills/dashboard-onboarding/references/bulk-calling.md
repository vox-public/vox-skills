# 파트 3: 대량발신하는 방법

고객이 다수의 대상에게 자동으로 전화를 걸 수 있도록 안내한다.

**진입 경로:** 사이드바 > 배포 > 발신 > **대량 발신**

## 3.1 프로젝트 생성

대량 발신은 **프로젝트** 단위로 관리한다. 프로젝트 = 발신 대상 리스트가 담긴 스프레드시트.

**방법 1: 직접 생성**

1. 대량 발신 페이지에서 **"+ 새 프로젝트 생성"** 클릭
2. 빈 스프레드시트가 열린다
3. 칼럼을 추가하고 (+), 발신 대상 데이터를 직접 입력한다
4. 필수 칼럼: **전화번호** (수신자 번호)

**방법 2: 파일 업로드**

1. 엑셀(.xlsx) 또는 CSV 파일을 준비한다
2. 새 프로젝트 생성 시 파일을 업로드하면 데이터가 스프레드시트에 자동으로 채워진다
3. 칼럼명이 프롬프트의 `{{변수명}}`과 일치하면 발신 시 자동으로 변수에 주입된다

프로젝트는 이름으로 검색할 수 있고, 여러 시트(탭)를 추가할 수 있다.

## 3.2 발신 설정 및 실행

1. 스프레드시트에서 발신할 **row를 체크박스로 선택**한다 (전체 선택 가능)
2. 오른쪽 상단 **"발신"** 버튼 클릭 → **전화 발신 설정** 모달이 열린다

모달 설정 항목:

1. **캠페인 이름** — 이 발신 건을 식별할 이름 (예: "3월 매장방문 필터링 1차")
2. **에이전트 선택** — 드롭다운에서 에이전트를 선택한다. 에이전트를 선택하면 **추출** 항목도 자동 표시된다.
3. **발신번호 / 표시번호** — 발신에 사용할 070번호와 수신자에게 표시될 번호를 각각 선택한다.
4. **동시 통화 수** — -/+ 버튼으로 조절 (기본값 5). 동시에 몇 명에게 전화를 거는지 설정한다. 숫자가 클수록 빨리 끝나지만 동시 통화 비용이 늘어난다.
5. **발송 시점** — "즉시 발송" 또는 "예약 발송" 선택. 예약 발송을 선택하면 날짜/시간을 지정한다.
6. **발신 허용 시간대** — 기본값 월~일 08:00~20:00. 이 시간대 밖에서는 발신이 일시 중단되고, 다음 허용 시간에 자동 재개된다.
7. **완료 알림 수신자** — 캠페인 완료 시 이메일 알림을 받을 사람. 이메일 주소를 태그로 추가한다.

설정 완료 후 **"확인"** 클릭 → 발신이 시작된다.

# 파트 1: 에이전트 변경하는 방법

고객이 에이전트의 동작을 직접 수정할 수 있도록 안내한다.

**진입 경로:** 사이드바 > 구축 > 에이전트 > 에이전트 목록에서 클릭

에이전트 상세 페이지는 크게 3영역으로 구성된다:

- **상단 바:** LLM 모델 선택(드롭다운 + 설정), TTS 음성 선택(드롭다운 + 설정), 언어 선택
- **왼쪽:** 프롬프트 편집기 (시스템 프롬프트 전문)
- **오른쪽 패널:** 도구, 지식 베이스, 통화 정보 추출, 발화 설정, 통화 설정, 보안 설정, 웹훅 설정 (아코디언)

## 1.1 프롬프트 수정

에이전트 상세 페이지 왼쪽의 프롬프트 편집기에서 직접 수정한다.

- 프롬프트는 Markdown 형식으로 작성한다 (# 역할, # 컨텍스트, # Variables, # 목표, # 말투/발화 스타일 등).
- `{{변수명}}` 형태의 런타임 변수를 사용할 수 있다. 보라색으로 하이라이트 표시된다.
- 하단에 **첫 메시지** 설정이 있다: "AI가 먼저 말함 (동적 메시지)" 또는 고정 메시지를 선택할 수 있다.
- 수정 후 상단 우측의 **저장** 버튼을 눌러야 반영된다. 저장하지 않고 페이지를 나가면 변경사항이 사라진다.

## 1.2 목소리(TTS) 및 음성 속도

**TTS 음성 변경:**

- 상단 바의 음성 드롭다운(예: "유진")을 클릭하면 사용 가능한 음색 목록이 표시된다.
- 음성 옆 설정(톱니바퀴) 아이콘을 클릭하면 상세 TTS 설정에 진입한다.

**발화 설정 (오른쪽 패널 > 발화 설정):**

- **반응속도:** 슬라이더로 조절 (느림 ↔ 빠름, 기본값 1.0). 낮추면 응답 지연 시간이 늘어난다.
- **말 끼어들기 허용:** 토글 ON/OFF. ON이면 고객이 말하는 중간에 AI가 발화를 중단한다.
- **키워드 강조:** 에이전트가 특정 키워드를 더 잘 인식하도록 키워드 리스트를 입력한다 (예: "복스에이아이, AI 에이전트").

## 1.3 툴 설정

오른쪽 패널 > **도구** 섹션을 펼친다.

- 현재 장착된 도구 목록이 표시된다 (예: `end_call` — 통화 종료).
- **"+ 추가"** 버튼을 클릭하면 빌트인 도구(end_call, transfer_call, send_sms, send_dtmf 등)와 커스텀 도구를 선택할 수 있다.
- 각 도구 카드의 **삭제(휴지통) 아이콘**을 클릭하면 도구를 해제할 수 있다.
- 도구마다 description이 있어 에이전트가 언제 해당 도구를 호출할지 판단한다.

## 1.4 통화정보 추출 설정

오른쪽 패널 > **통화 정보 추출** 섹션을 펼친다.

- 통화 종료 후 통화 내용에서 자동으로 추출할 정보를 설정한다.
- 기존 추출 필드가 카드로 표시된다 (예: "신용카드*여부", "방문가능*여부"). 각 카드에 필드명과 추출 설명이 있다.
- **"+ 추가"** 버튼으로 새 추출 필드를 만든다: 필드명과 추출 기준(description)을 입력한다.
- 추출된 데이터는 **통화 기록** 테이블에 칼럼으로 자동 표시된다 (call-data.md 4.3 참조).

## 1.5 버전 배포

에이전트 상세 페이지 오른쪽 상단의 **배포** 버튼을 클릭한다.

- **에이전트 배포 기록** 모달이 열린다. 이전 배포 이력을 확인할 수 있다.
- 하단 **"설정"** 버튼을 클릭하면 현재 에이전트 상태를 프로덕션 버전으로 배포한다.
- 배포하지 않은 상태에서는 현재 편집 중인 버전(draft)으로 동작한다.
- 대량 발신 시 에이전트 버전을 선택할 수 있다 (current 또는 배포된 특정 버전).

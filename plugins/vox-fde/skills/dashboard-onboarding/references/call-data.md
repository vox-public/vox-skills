# 파트 4: 통화 데이터 보는 방법

발신 결과와 통화 내용을 확인하고 활용하는 방법을 안내한다.

## 4.1 대량발신 기록

**진입 경로:** 사이드바 > 배포 > 발신 > **대량 발신 기록**

캠페인별로 발신 결과를 한눈에 볼 수 있다:

- **이름**: 캠페인 이름
- **상태**: 완료/진행중
- **예약 | 발신 | 수신**: 각 단계별 건수 (예: 1 > 1 > 0 = 예약 1건, 발신 1건, 수신 0건)
- **비용**: 사용된 크레딧
- **발신번호 / 표시번호**: 사용된 전화번호
- **시작일시**: 캠페인 시작 시간

상단에 **날짜 필터**(캘린더 아이콘), **캠페인 검색**, **다운로드**(↓ 아이콘)가 있다.

## 4.2 통화 기록

**진입 경로:** 사이드바 > 모니터링 > **통화 기록**

개별 통화 단위로 상세 데이터를 조회한다:

- **통화 일시**: 통화가 이루어진 날짜와 시간
- **에이전트**: 통화에 사용된 에이전트 이름
- **버전**: 에이전트 버전
- **통화 시간**: 총 통화 길이 (분:초)
- **통화 타입**: 웹/전화
- **비용**: 소모된 크레딧
- **연결 종료 사유**: "에이전트 종료" (AI가 end_call) 또는 "사용자 종료" (고객이 끊음)
- **유저 감정**: 긍정/중립/부정 (AI가 통화 내용 기반으로 자동 분류)

통화 row를 클릭하면 상세 페이지에서 녹음 재생, 트랜스크립트 확인이 가능하다.

## 4.3 통화정보 추출 칼럼 확인

통화 기록 테이블을 오른쪽으로 스크롤하면 에이전트에 설정한 **통화정보 추출 필드**가 칼럼으로 표시된다.

예를 들어 에이전트에 "신용카드*여부", "방문가능*여부" 추출을 설정했다면, 통화 기록 테이블에 해당 칼럼이 나타나고 각 통화별로 추출된 값(true/false/-)이 표시된다.

## 4.4 날짜 및 필터 사용

통화 기록 상단 우측의 도구:

- **캘린더 아이콘** — 날짜 범위를 선택해서 특정 기간의 통화만 조회
- **필터 버튼** — 필터 패널이 열리며 다음 항목으로 필터링 가능:
  - Call ID, 에이전트, 발신번호, 표시번호, 수신번호, 연결 종료 사유, 통화 유형, 유저 감정
- **칼럼 설정 아이콘(격자)** — 테이블에 표시할 칼럼을 선택/해제
- **새로고침 아이콘** — 데이터 갱신

## 4.5 엑셀 다운로드

통화 기록 상단 우측의 **다운로드(↓) 아이콘**을 클릭하면 현재 필터가 적용된 상태의 통화 데이터를 엑셀 파일로 내보낸다.

다운로드 파일에는 테이블에 표시된 모든 칼럼(통화정보 추출 칼럼 포함)이 포함된다.
필터와 날짜 범위를 먼저 설정한 후 다운로드하면 원하는 데이터만 추출할 수 있다.

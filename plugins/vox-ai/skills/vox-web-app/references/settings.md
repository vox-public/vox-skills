# 설정 (Settings) 영역

vox.ai 웹 앱의 **설정** 그룹은 조직, 결제, 멤버, API, 웹훅, 개인 프로필 등 운영 전반을 다룬다.

루트: `/dashboard/{orgId}/settings` (카테고리 선택)

---

## 1. 워크스페이스 (`/settings/workspace`)

조직 기본 정보.

- **이름** 수정
- 로고/아이콘 (있다면)
- 생성일
- 조직 ID 표시 (API/MCP 호출 시 필요)

---

## 2. 결제 (`/settings/billing`)

구독 플랜, 결제 수단, 청구 내역 관리. UI에서만 가능한 영역.

### 2.1 현재 플랜
- 활성 구독 플랜 (Starter / Pro / Enterprise 등)
- **플랜 변경** 버튼 → 플랜 선택 모달 → 결제 처리

### 2.2 결제 수단
- 등록된 카드 목록
- **결제 수단 추가** → 카드 등록 폼 (토스페이먼츠 등 외부 PG 연동)
- 기본 결제 수단 설정, 삭제

### 2.3 청구 내역
- 월별/기간별 청구서 목록
- 각 청구서 상세 (항목별 금액)
- **PDF 다운로드**
- 환불 요청 버튼 (정책 내 건만)

### 2.4 사용량
- 현재 달 크레딧 사용 현황 차트
- 잔여 크레딧

> 가격/플랜 정책 상세 설명은 `vox-docs` MCP의 pricing 페이지 참조.

---

## 3. 멤버 (`/settings/member`)

조직 멤버 초대와 권한 관리.

- 등록된 멤버 목록 (이메일, 권한, 가입일)
- **멤버 초대**: 이메일 입력 → 권한 선택(owner/admin/member) → 초대 메일 발송
- 멤버 제거, 권한 변경

> 초대 메일 발송과 권한 변경은 UI에서만 가능.

---

## 4. API 키 (`/settings/api-key`)

vox.ai API를 외부 시스템에서 호출할 때 사용할 키 관리.

- 등록된 키 목록 (prefix만 표시, full key는 생성 시 한 번만)
- **+ 새 키 생성** → 이름 입력 → 키 발급 → 복사 안내
- 재생성 / 삭제

> 키는 발급 시점에만 full value를 볼 수 있다. 분실 시 재생성 필요.

---

## 5. 웹훅 (`/settings/webhook`)

vox.ai 시스템 이벤트(통화 종료 등)를 외부로 전송하는 웹훅 등록.

- **Organization webhook**: 조직 전체 이벤트
- **Webhook key** 발급 (서명 검증용)
- 테스트 발송 버튼으로 웹훅 수신 확인

> 에이전트 단위 웹훅은 에이전트 상세 > 오른쪽 패널 > 웹훅 설정에서 별도 관리 (build.md 1.3 참고)

---

## 6. SMS (`/settings/sms`)

SMS 발송 이력과 설정.

- 발송 내역 (`send_sms` 도구로 보낸 메시지)
- 잔액 / 충전 내역
- 발신자 등록 (개별 텍스트 메시지에서 사용할 표시 발신자)

---

## 7. 프로필 (`/settings/profile`)

개인 계정 정보.

- 이름, 이메일, 프로필 이미지
- 비밀번호 변경
- 2차 인증(MFA) 설정
- 언어/시간대

---

## 일반 원칙

- **UI 전용**: 결제, 멤버 초대, 프로필, 세션 관리는 보안상 MCP 도구로 제공되지 않음
- **딥링크 없음**: 설정 영역은 대부분 라우트로만 이동하며, 특별한 쿼리 파라미터로 다이얼로그를 자동 오픈하지 않음
- **권한**: 결제/멤버/API 키는 owner/admin 권한 필요. member 계정은 워크스페이스/프로필만 수정 가능
